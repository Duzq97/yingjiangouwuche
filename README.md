# 微信小程序：装机硬件清单

这是一个可导入微信开发者工具的小程序工程，并附带网页预览端。

## 功能

- 扫码进入小程序后展示硬件清单主页面。
- 分类包括：CPU、主板、显卡、散热器、电源、内存、硬盘、机箱、杂项。
- 商品数据来自同一份 CSV：`miniprogram/data/hardware.csv`。
- CSV 字段：硬件分类、硬件名称、硬件描述、硬件价格、硬件图片。
- 每个商品支持加入购物车，购物车支持数量增减和总价统计。
- `web-preview` 用于浏览器预览开发进度和 CSV 内容。

## 使用方式

1. 用微信开发者工具导入 `miniprogram` 目录。
2. 将 `project.config.json` 里的 `appid` 替换为你自己的小程序 AppID。
3. 修改 `miniprogram/data/hardware.csv` 即可配置分类和商品内容，保存时请选择 UTF-8 编码。
4. 硬件图片如果使用线上 URL，正式发布前需要在微信公众平台配置图片域名白名单。
5. 上传版本后，可在微信公众平台生成小程序码或普通二维码，用于扫码进入。

## 网页预览

在项目根目录启动静态服务，然后打开：

```bash
python -m http.server 4173 --bind 127.0.0.1 --directory .
```

访问：

```text
http://127.0.0.1:4173/
```

后台配置单页面：

```text
http://127.0.0.1:4173/admin.html
```

网页预览版的配置单会保存在浏览器 `localStorage` 中，适合本地演示。正式小程序跨用户查看提交记录时，需要接入微信云开发数据库或自己的服务器接口。
